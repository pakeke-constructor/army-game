# ===============
# Prestige:
# ===============

We can't keep going "up".
Once there's a bunch of shit on screen, we need a way to "reel it back in".  

Since there is finite-space on the upgrade-tree,  
I propose the following:    
Each "prestige" has a different upgrade-tree.

Many upgrades span across multiple prestiges.  

What's cool is that every time the player prestiges, they discover and unlock new upgrades; so it always feels "fresh".

## CATEGORICAL PRESTIGE UPGRADES. 
To start with, we group upgrades together, in "categories".
Then, you can choose to prestige for a particular category.

Prestige doesn't wipe the ENTIRE progress of the player;  
instead, it just wipes all the upgrades in a category; and *replaces them with new ones.*  
(make sure player is refunded for money spent tho; the prestige upgrade should be like 50x-100x more expensive than any other upgrade in the category)  

This is very elegant, since we can reuse map-space.  
Also it makes progression a lot easier for us to handle.  


## PRESTIGE EXAMPLE:
PRESTIGE: Resets mining bonuses, decreases mining area
--> Increases ALL harvesting by 2x
