


# THEMING:

Whatever the case, theming *MUST BE CONSISTENT.*  



## THEME DRAFT IDEA: Cat-themed resource-farming game.

- Cat-like avatars for players
- Cat-theming for upgrades
- Fantasy-like setting for the world

^^^ In general, the theme shouldn't take itself too seriously.  
Feel free to make up new words/concepts.


IDEA: For tokens, we should have cats as tokens.  
(Less early-game... more cats late-game)
E.g. copycat-sprite, crappy-cat-sprite, etc etc.  
I think that this would add a lot of "heart" to the game.
=---  
Besides, the number of visual variations of cats is huge.





# NAMING:
Try to avoid generic names for stuff! We don't want it to be boring.  
Make it more *exciting!* Give the world some character :)   
EG:  
- box -> litterbox
- sword -> kitten-knife
- explosive-barrel -> cat-bomb
- Magic dust -> Paw shimmer
- Fluffy ball -> Whisker fluff

Ask chatGPT for ideas if you are running low on creativity.
