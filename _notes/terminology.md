

## TERMINOLOGY:


Zone: The current "map" you are in. Start at Zone-1
Timer: The "countdown timer" until bossfight and map-reset (zone-increase)


Node: the world objects you move between at the cost of time and food
Enemy-Node: Contains enemy-encounter
Elite-Node: Contains stronger enemy-encounter (better reward)
Village-Node: Contains village, shop
Shrine-Node: Contains a blessing
Event-Node: Contains a random event
Portal-Node: Warps player to a new randomized map
Empty-Node: Contains nothing (just a place to traverse)


RESOURCE: A value that presists across combat
Food:  resource. Consumed when moving.
Money:  resource. used to buy stuff
Mana:  resource. used to cast spells
Health:  each unit has health. Persists across combat.

Map: a collection of nodes and decorations


Commander: Player has 1 commander in their army. Each commander has different starting blessing. Can unlock more.

BattleField: where the battles are hosted, where objects are stored

Unit: A singular individual (eg an archer)
Squad: A collection of units (eg 5 x archers)
Army: A collection of squads

Blessing: Permanent global buff that you get during runs (chests, shop, etc)

Perk: Buff/effect that applies to a squad.
